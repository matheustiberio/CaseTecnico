# Case técnico - PBTech

### O seu objetivo é fazer uma WebApi desenvolvida em .net 6.0 para registros de clientes que deverão informar o nome completoe e-mail para receber informações.

### Para isso você vai aplicar o conceito de CRUD

a. Create;
b. Read;
c. Update;
d. Delete.

### Você deverá entregar é uma WebApi documentada com o Swagger que tenha as seguintes funções:

1. Cadastro de cliente informando o nome completo e e-mail;
2. Consulta de todos os clientes da base;
3. Consulta de cliente através do e-mail;
4. Atualização de nome e e-mail (você deve fazer um método que atualize o nome e e-mail de um cliente registrado);
5. Exclusão de um cliente através do e-mail.

### Requisitos técnicos:
1. Desenvolver em .Net 6.0;
2. Utilizar base de dados local;
3. Criar testes de unidade.
